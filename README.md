# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 기능 요구사항
- ~~컴퓨터가 1에서 9까지 서로다른 임의의 수를 선택하는 기능~~
- 플레이어가 3자리의 수를 입력하는 기능
- 플레이어가 제시한 숫자와 컴퓨터의 숫자가 일치하는지 판별
- 일치한다면 게임 종료
- 일치하지않는다면 ERROR 출력하며 계속 진행
